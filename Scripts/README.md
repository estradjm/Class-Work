# My Custom Vim Configuration

Support for Git version control within Vim, directory browsing with filetype icons, status line with glyphs, syntax highlighting, key mappings for running Python with F4 key, code folding and block commenting shortcuts, syntax error checking (linter), template color scheme changing with F9 and F10, and tagbar (toggled with F2 with key mapping), using Terminator for screen management/splitting.
